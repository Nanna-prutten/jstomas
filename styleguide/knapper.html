Knapper

Denne side viser både knapper og boxes som bliver brugt i APPen. De er her for at give standart guide, til vores projektgruppe og skabe sikkerhed i om tingene er rigtige. De skal nogle have skiftede farve og andet men dette er den helt standart.

HTML:

<button class="button button1">start/tilbage</button>

css:

.button { background-color:#73AE5F; border-radius: 12px; border: 1px solid #fffff; padding: 14px, 32px; width: 200px; height: 15px; text-align: center; color: #ffffff; font-size: 16px; font-family: Noto Sans HK; padding: 25px; transition-duration: 0.4s; } .button1:hover { background-color: #4CAF50; color: white; }

forside infobox.

HTML:

<div class="transbox"> <div class="text"> <p>forside infobox.</p> </div> </div>

css:

div.transbox { margin: 30px; background-color: #000000; opacity: 0.5; border-radius: 18px; border: 1px solid #ffffff; padding: 15px; width: 200px; height: 30px; } div.text { margin: 5%; color:white; text-align: center; font-size:10px; font-family:Noto Sans HK; }
